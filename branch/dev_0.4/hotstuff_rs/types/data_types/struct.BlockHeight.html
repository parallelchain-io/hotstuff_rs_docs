Struct BlockHeight

source 
pub struct BlockHeight(u64);
Expand description

Height of a block in the block tree.

Starts at 0 for Genesis Blocks (blocks that contain the genesis_pc), and increases by 1 for every subsequent “level” of blocks connected by justify links.
